# Contributing to Lesson Plan API

Thank you for your interest in contributing to the Lesson Plan API!

## Code Sample Note

This project is primarily a **code sample** for the MLH Fellowship application. However, suggestions and feedback are welcome!

## How to Provide Feedback

1. **Open an Issue**: Describe your suggestion or bug report
2. **Propose Changes**: Feel free to fork and submit a PR
3. **Discussion**: Start a discussion about potential improvements

## Development Setup

See [QUICKSTART.md](docs/QUICKSTART.md) for setup instructions.

## Code Standards

- Follow PEP 8 style guide
- Use type hints where possible
- Write tests for new features
- Update documentation as needed

## Testing

All contributions should include tests:

```bash
pytest tests/
```

## Questions?

Feel free to open an issue for any questions!
